How to Download SpacEscape

  1. Navigate to Google Play Store app within your Android device (You must have an existing google account to sign into or create one using an existing email.)
  2. Search for SpacEscape (see game icon below), or use this URL for downloading: Download
  3. Tap/Click on the install button.
  4. Wait until installation is complete.
  5. You can find the app by clicking on “open” in the Google Play Store under your app or on your home screen.
  6. Input the User ID your teacher provided to you and enjoy the game!
  7. Or apply for a user ID here:User ID Application We will generate an ID for you within 72 hours if your application is approved
